# Fashion Store E-Commerce Website

A simple multi‑page fashion store built with **HTML, CSS, and JavaScript**.  
This project was created as part of an internship task to demonstrate front‑end web development skills.

---

## 🚀 Features
- **Multi‑page layout**: Home, Shop, About, Contact
- **Dynamic product listing**: Products load from JavaScript with images, prices, and star ratings
- **Interactive buttons**: "Add to Cart" alerts for user engagement
- **Responsive design**: CSS grid and hover effects for a modern look
- **Styled UI**: Gradient backgrounds, navigation bar, hero section, and contact form

---

## 🛠️ Tech Stack
- **HTML5** for structure
- **CSS3** for styling and layout
- **JavaScript (ES6)** for dynamic product rendering and interactivity
